The Cornerstone Framework was designed to be the most basic of basic frameworks. For developers who want to do all of their own styling, but just need the basic features taken care of quickly and efficiently to design for a responsive platform. You wont find any styling included in this framework, that is for you. Build your websites exactly how you want without the heavy lifting of structure to worry about. No more overriding styling that comes preloaded with most frameworks. Welcome to Cornerstone, I hope you enjoy using this as much as I do.

Slideshow

Implementation: There is a provided slideshow module that comes with the Cornerstone framework. It was built by Pixedelic - Pixedelic.com. To use this slideshow inside of your website, use the example code that is included below. Rename the path to the images to wherever you are storing your photos.

<div id="slideshow">
  <div class="camera_wrap" id="camera_wrap_1">
    <div data-thumb="images/slides/thumbs/leaf.jpg" data-src="images/slides/leaf.jpg"></div>
    <div data-thumb="images/slides/thumbs/road.jpg" data-src="images/slides/road.jpg"></div>
    <div data-thumb="images/slides/thumbs/leaf.jpg" data-src="images/slides/leaf.jpg"></div>
    <div data-thumb="images/slides/thumbs/bridge.jpg" data-src="images/slides/bridge.jpg"></div>
  </div>
</div>

Framework

Implementation: Using the tags that are reserved for use by the Cornerstone Framework you can quickly and easily build a structure for your website without hassle. The <div class="container"> </div> tag will create your initial container div. Inside of that div you will place a <div class="row"> </div> tag that will serve as your horizontal row. Inside of the row class div, you will then place your specificed column width by using the tag <div class="column100"> </div> (or what ever size that you specify. Make sure that your columns add up to a width of 100 wide or less (ex: column75 + column 25). Inside of the column div you then have the option of using a supplied spacer div, <div class="column-box"></div>. This div creates a nice padding between your content in each of the columns.


You will also notice that this framework is fully responsive, grab the edge of the screen and shrink the browser to see how it works.

<div class="container">
  <div class="row">
    <div class="column75">
      <div class="column-box">
        your content goes here
      </div>
    </div>
    <div class="column25">
      <div class="column-box">
        your content goes here
      </div>
    </div>
  </div>
</div>

Additional Structure Tags

Implementation: A list of additional structure tags and their uses.

<header></header> - contains the header portion of the document
<footer></footer> - contains the footer portion of the document
<div id="wrapper"></div> - wrapper div that contains the entire body of the document.
<div id="title"></div> - wrapper div that contains the title of the webpage in the form of image or text.
<div id="main"></div> - div that contains the main content of the body

Main Navigation

Implementation: Included is a very simple version of a horizontal navigation menu that includes a sub menu. You will need to style the visuals of the menu, but the function should work in its entirety.

<nav>
  <ul>
    <li><a href="#">Home</a></li>

Images

Implementation: This includes the full-width-image class that will make any static photograph fill the width of whatever size div that it is inside of. Also makes that image fully responsive.

<img src="images/slides/bridge.jpg" class="full-width-image" />
